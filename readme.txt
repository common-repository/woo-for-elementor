== Description ==

Add new Woocommerce Widgets that are specifically designed to be used in conjunction with the Elementor Page Builder.

== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'WooCommerce For Elementor'
3. Activate WooCommerce For Elementor from your Plugins page.
4. Go to "After activation" below.

=== Manually ===

1. Upload the `woo-for-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the WooCommerce For Elementor plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

=== After activation ===

1. Some Widget added on Elementor Pagebuilder, let's use them.
